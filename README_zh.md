## 📖 项目简介

这个微信小程序旨在为家庭用户提供一个简单、便捷的记账工具。无需下载额外 App，通过微信即可随时记录收入与支出，支持按类别、日期和关键词进行筛选，并能计算总收支和结余。

## ✨ 核心功能

- **快速记账**：支持记录收入和支出，并可加以备注。
- **灵活筛选**：
    - 按时间范围（自定义日期区间）筛选记录。
    - 按类型（收入/支出）筛选记录。
    - 按备注关键词筛选记录。
- **数据持久化**：使用微信本地存储，确保数据在小程序关闭后依然保留且不上传云端。

## 🏗️ 技术架构

- **前端框架**: 原生微信小程序 (WXML, WXSS, JavaScript)
- **数据存储**: 微信小程序本地存储 API

## 🚀 快速开始

1.  **安装微信开发者工具**
    - 访问 [微信小程序官方文档](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装。

2.  **获取项目代码**
    - Clone 或 Download 此仓库到本地。
    ```bash
    git clone https://github.com/LiQVQ/family-accounting-WeChat-Mini-Program.git
    ```

3.  **导入项目**
    - 打开微信开发者工具，选择 "导入项目"。
    - 选择项目文件夹路径，并填入 AppID（如果没有可使用测试号，我也是测试号写的🥳🥳🥳）。
